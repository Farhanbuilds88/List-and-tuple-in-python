# List-and-tuple-in-python

This repository contains my practice programs on Python Lists and Tuples.  
It covers basic to intermediate concepts including indexing, slicing, mutability, list methods, tuple operations, and simple logical tasks.

---

## 📌 Topics Covered

### 🔹 Python Lists
- Creating lists
- Accessing elements using indexing
- Finding length using `len()`
- Storing multiple data types in a single list
- List mutability (modifying elements)
- List slicing
- List methods:
  - `append()`
  - `sort()`
  - `reverse()`
  - `insert()`
  - `remove()`
  - `pop()`
  - `copy()`
- Checking palindrome using list reverse logic
- Taking user input and storing values in list

---

### 🔹 Python Tuples
- Creating tuples
- Single element tuple
- Accessing elements
- Tuple immutability
- Tuple methods:
  - `count()`
  - `index()`
- Difference between list and tuple

✔ Lists are mutable (can be changed)

✔ Lists can store multiple data types  
✔ Sorting in ascending and descending order  
✔ Reversing a list  
✔ Checking palindrome using list copy and reverse  
✔ Taking user input and storing in a list
